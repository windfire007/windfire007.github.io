这是一份自用软件的列表，也就是目前电脑上装的一些软件，不包括系统自带的。留个备份，万一哪天电脑突然紧急重装了，至少还知道该装啥软件。电脑上所有的软件都是正版，包括windows。

---

[**QQ**](http://im.qq.com/pcqq/)

```
聊天软件，已经成为生活的必需品了。我最最最主要的社交软件，以及最好的截图软件。
```

---

[**微信**](http://weixin.qq.com/cgi-bin/readtemplate?t=win_weixin&lang=zh_CN)

```
同样的聊天软件，反正我是不咋用，用都是被逼的。如果有可能，我才不想用微信呢。
```

---

[**搜狗输入法**](http://pinyin.sogou.com/)

```
输入法这东西，中文还是中国人做的靠谱，虽然还是有点臃肿了。支持双拼，多平台同步，输入算是比较只能，虽然对双拼没啥优化。之前用的qq拼音的替代品，反正qq拼音的团队都归搜狗管了。
```

---

[**IT之家**](http://m.ruanmei.com/)

```
当年用wp时留下来的习惯，现在也一直在上面看看it新闻，虽然这地方和大部分网站也没啥区别。
```

---

[**北大网关**](https://its.pku.edu.cn/download_ipgwclient.jsp)

```
学校里登录上网账号用的，不过说好发新版的啊，到现在还是旧版。
```

---

[**Office 365**](https://products.office.com/zh-cn/office-365-personal)

```
Office 365的个人版，包含Word、Excel、PowerPoint、OneNote、Outlook、Publisher、Access。最主要是送1TB的OneDrive储存空间。价格399/年，属于完全可以接受的范围，用了这么多年，还是应该支持正版。虽然我有学校提供的免费的教育版，但是属于被阉割版，也就不太想用了。
```

[Word](https://products.office.com/zh-cn/word)

```
最强大的文字编辑软件，没有之一。
```

[Excel](https://products.office.com/zh-cn/excel)

```
最强大的电子表格软件，可惜我只会皮毛。平时用来做一些概率实验。
```

[PowerPoint](https://products.office.com/zh-cn/powerpoint)

```
最强大的幻灯片软件，我也只会皮毛，不过微软现在也加入了很多模板，可以直接改模板了。
```

[OneNote](https://www.onenote.com/)

```
一款同步笔记软件，也可以用来记日记。我平日用来写日记，当备忘录记一些东西。不过由于网页版是依附于onedrive，所以用网页版需要翻墙。
```

[Outlook](https://products.office.com/zh-cn/outlook/)

```
最强大的邮箱客户端，我主要用来收邮件看邮件，一般不用来发邮件，发邮件还是网页端显示靠谱。不过这玩意存放删发邮件的地方可能和网页端不太一样，而且网页端删邮件可能又会被outlook同步回来。
```

[Publisher](https://products.office.com/zh-cn/publisher)

```
据说是排版用的，和我没多大关系。
```

[Access](https://products.office.com/zh-cn/access)

```
数据库……没理解和excel的区别，反正我也不用。
```

[OneDrive](https://onedrive.live.com/about/zh-cn/download/)

```
Office365送的1TB空间，不用白不用，win自带一个同步盘，教育网环境下网速极其快。平时用来上传下载东西，比下面那个uwp版要靠谱。同步盘可以选择同步的文件夹，这点还是很方便的，专门弄一个和mac同步，以及存放需要本地用到的东西。退出账号重新登录可以更改本地onedrive的位置。
```

---

[**OneDrive UWP**](https://www.microsoft.com/zh-cn/store/p/onedrive/9wzdncrfj1p3)

```
由于网页版的onedrive被墙了，而且win自带的同步盘不能查看文件目录，所以用uwp来查看网盘，当然对我而言，也就是相当于网页版的功能，用来查看文件或者移动删除文件。
```

---

[**Sway**](https://sway.com/)

```
之前win10自带的uwp应用，不过现在好想又不是系统自带的了。这个类似于一个网页端的展示软件，见单几部就能制作出一个html的展示出来，不过我也不用。
```

---

[**Adobe Acrobat Reader DC**](https://get.adobe.com/cn/reader/)

```
adobe的免费pdf阅读软件，也是最流行的的pdf阅读软件，由于是免费版，没有一些编辑功能，不过只要能看对我来说就够了。可以关联onedrive直接看网盘内容。
```

---

[**Typora**](http://www.typora.io/#windows)

```
就是我现在用来写文章的软件，一款markdown的编辑器，可以实现实时预览格式效果，不是双屏显示那种，是同屏即时显现。很多格式都能自动化，简单易于新手。不过有些地方不是自己能控制的，必须用代码模式才能改。
```

---

[**Google Chrome Dev**](http://www.chromium.org/getting-involved/dev-channel)

```
chrome是最好的浏览器，也是使用人数最多的浏览器，大部分国产浏览器都是chrome套壳。我在win上一直用的是开发(dev)版，不过dev版的bug确实多啊，毕竟开发版，还是能忍受这些bug的。用chrome还是需要一个gmail账号的，可以同步各种事，不同设备的浏览记录也可以同步，而且同步也不需要翻墙，虽然偶尔会抽风。支持的插件也各种各样。
```

[OneNote Web Clipper](https://chrome.google.com/webstore/detail/onenote-web-clipper/gojbdfnpnhogfdgjbigejoaolejmgdhk)

```
chrome插件，可以用来把网页内容发送到onenote上。可以选择整个网页截图、局部截图、保存文字、单单保存标题链接四种模式。不过我用的不太多。
```

[Save to Pocket](https://chrome.google.com/webstore/detail/save-to-pocket/niloccemoadcdkdjlinkgdfekeahmflj)

```
chrome插件，可以将网页保存到pocket上。pocket是个稍后阅读的软件，属于备忘类型吧。我最常用除了备忘，就是保存链接后在不同设备上同步。
```

[Awesome Screenshot: Screen capture, Annotate](https://chrome.google.com/webstore/detail/awesome-screenshot-screen/nlipoenfbbikpbjkfpfillcgkoblgpmj)

```
chrome插件，网页截图和备注用的。我几乎用不上这玩意，但是这插件可以截图整个网页，偶尔还有点用处。不过onenote也可以……是时候把这插件删了。
```

---

[**Avira Antivirus Pro**](http://www.avira.com/en/downloads)

```
小红伞，一个德国的杀毒软件，有免费版。收费版增加了几个没啥用的功能，免费版的杀毒功能就很齐全了。本来我就是裸奔党也不用杀软的，我只是因为听说小红伞打折而已，然后就一下买了三年，图个安心，不为别的。
```

[爱红伞](http://www.iavira.com/)

```
小红伞的中文代理网站，序列号就是在这个网站上买的，当时年中促销。在这个网站上可以下载中文版的小红伞。
```

---

[**XnView**](http://www.xnview.com/en/xnview/)

```
免费的看图软件，还提供很多工具。基本的图片修改都提供，截图、格式转换、分辨率修改、旋转、调色等等，还提供文件批量改名功能。支持gif一帧一帧查看，ico如果有不同的尺寸也可以分别查看。
```

---

[**AIMP**](http://www.aimp.ru/index.php?do=download)

```
俄国出品的本地音乐播放软件，高品质的音乐软件，反正我没好耳机，也就是想要个单纯的音乐播放器而已。提供标签编辑和格式转换等功能。我主要用它来播放LoveLive的歌曲，不过也很久没听了啊。
```

---

[**网易云音乐**](http://music.163.com/#/download)

```
国内快速起步的一个在线音乐软件，凭借着认真对待各种平台而圈了一批粉。我主要用于没事随机听音乐用，听个响而已。评论区也挺不错的。
```

---

[**PotPlayer**](http://potplayer.daum.net/?lang=zh_CN)

```
强大的本地视频播放器，也有很强的定制功能，支持连续截图。因为qq影音几乎被腾讯抛弃了，所以寻求的替代品，使用起来我很满意。
```

---

[**Bandizip**](http://www.bandisoft.com/bandizip/cn/)

```
免费的压缩软件，替代各种国产流氓的。最大特点在于可以选择编码，来防止文件名乱码的问题。不过不支持添加注释、计算hash这些功能,支持的压缩格式倒是挺多的。不过为了在mac下解压不乱吗，还是建议在设置里把文件名默认utf-8编码给勾上。
```

---

[**µTorrent**](http://www.utorrent.com/intl/zh_cn/)

```
一款bt客户端，也终于让我明白了什么叫做bt下载，种子到底该是什么样子。作为纯粹的bt客户端来说是很不错的。不过我用的时候，不知为何磁力链不能使用，这就比较麻烦了。磁力链的下载，最好用的还是qq旋风
```

---

[**EagleGet**](http://www.eagleget.com/cn/)

```
一款下载软件，纯粹的下载软件，所以不支持bt。也可以拔取在线视频，不过偶尔会失败。虽然说主打的是不流氓不限速，不过自从用了eagleget后，我的chrome就下载不了任何文件了……不过我也不知道是不是eagleget的问题，也可能是dev版的bug。自从有次qq旋风离线抽风不让用了之后，便找到的替代品，而且现在qq旋风的离线也几乎用不了了。
```

---

[**Steam**](http://store.steampowered.com/about/) 

```
这是一个诈骗软件，会勒索你的钱包，一定要慎重。言归正传，steam作为全球最大的游戏平台，确实养成了我是用正版的习惯，也给童年很多游戏补了一份正版。不过说实话，还是没太多的时间去玩游戏，到头来我玩的不是游戏，而是sbeam。
```

---

[**战网客户端**](https://www.battlenet.com.cn/account/download/index.xml?show=bnetapp)

```
暴雪爸爸的游戏平台，玩守望先锋用的。也可以当聊天软件用……
```

---

[**Uplay**](https://uplay.ubi.com/)

```
育碧的游戏客户端，俗称playU。由于育婊的游戏steam都有卖，所以只是个steam附赠品，而steam提供的uplay版本太老用不了。使用时经常需要翻墙，主要用来玩情怀的英雄无敌系列。
```

---

[**Minecraft**](https://minecraft.net/zh-hans/download/)

```
著名的沙盒游戏，不过这个3D效果我玩久了是真的头晕啊。而且不插电源简直卡的不能玩。
```

[Minecraft UWP](https://www.microsoft.com/zh-cn/store/p/minecraft-windows-10-edition-beta/9nblggh2jhxj)

```
买pc版会送uwp版，主要是uwp版支持手柄。uwp更类似于pe版，只要用电池不卡。
```

---

[**GeForce Experience**](http://www.geforce.cn/geforce-experience)

```
nvidia出的显卡更新、游戏优化软件，玩游戏必备。最近的版本不登录账号还不能用了。。。虽然说流传N卡都是负优化，不要更新驱动，不过讲道理我还是愿意更新的……
```

---

[**VMware Workstation Player**](https://my.vmware.com/cn/web/vmware/free#desktop_end_user_computing/vmware_workstation_player/12_0)

```
vm免费的虚拟机软件，现在vm的软件已经合并了，免费用就是player，付费就是pro。player也加入了创建功能，虽然没有快照，不过功能已经很够用了。我主要用来装windows insider，以及一些不想装到电脑上的软件。
```

---

[**Everything Search Engine**](http://www.voidtools.com/)

```
win下的所搜本地文件的神奇，搜索速度十分之快，第一次打开时相对会慢一点。卸载软件之后，配合everthing删除残留效果很好。
```

---

[**WizTree**](http://antibody-software.com/web/software/software/wiztree-finds-the-files-and-folders-using-the-most-disk-space-on-your-hard-drive/)

```
一款可以查看文件夹大小的软件。win最让我觉得不足的就是，无法查看文件夹的大小。以前装的软媒魔方对我最大的用处也就是可以查看文件夹大小，这软件也是魔方的替代品。查看每个目录的大小，可以很直观的了解文件的占用，而且本软件还能查看系统显示不出来的文件，虽然没啥用。
```

---

[**OpenVPN**](https://openvpn.net/index.php/open-source/downloads.html)

```
一个翻墙的软件，不过只是协议的客户端，具体的配置还需要找人买，或者自己搭。这是真的vpn，也就是说会安装虚拟网卡。
```

---

[**ShadowsocksR-win**](https://github.com/breakwa11/shadowsocks-csharp/releases)

```
翻墙的软件，同样也只是个协议的客户端，具体的配置需要找人买。不过这不属于vpn，所以不需要装虚拟网卡。
```

---

[**WD Security**](http://support.wdc.com/downloads.aspx?lang=cn)

```
给西部数据硬盘加密用的软件。加密后win和mac都可以用，linux不能使用。
```

---

[**小丸工具箱**](http://maruko.appinn.me/)

```
一个视频压制软件，提供压视频的各种常用功能。对我来说主要当做MKVToolnix的壳，合成mkv或者抽取mkv轨道，其余的功能也不太常用。
```

---

[**Java**](https://www.java.com/zh_CN/download/manual.jsp)

```
说实话我也不知道为啥安装了java。。。没啥用，chrome现在已经默认禁止java了，minecraft自带java也不需要额外安装，所以到底是为啥装了java呢？
```
